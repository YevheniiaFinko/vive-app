## 💡 Ідея продукту — VIVE

**Що це:** AI wellness-компаньйон для жінок 35+ у форматі веб-додатку (зараз MVP як React-файл, потім мобільний додаток).

**Проблема яку вирішує:** Жінки після 35 відчувають зміни в енергії, гормонах, сні — але лікарі дають загальні поради. AI може давати персоналізовані інсайти на основі щоденних даних і фази циклу.

**Ніша:** FemTech + Longevity для жінок 35+. Конкуренти (Flo, Clue) роблять трекінг — ми робимо **AI-персоналізацію + longevity протоколи**.

**Ринок:** FemTech $59B у 2026, зростання до $246B до 2035. CAGR 16%.

---

## 🗺️ Дорожня карта продукту

- **Зараз (MVP):** React веб-додаток — тестується особисто + 10-30 бета-юзерів
- **Місяць 1-2:** Зібрати фідбек, показати 10-30 жінкам, запустити монетизацію ($9/міс)
- **Місяць 3-4:** Додати Claude/OpenAI API для справжніх AI-відповідей
- **Місяць 6-8:** No-code мобільний додаток (Bubble або Glide)
- **Місяць 12-18:** Повноцінний React Native / Flutter додаток з розробником

## 📱 Що вже побудовано (файл: vive-app.jsx)

Повноцінний React MVP з 5 екранами:

### Екран 1 — Welcome
Лендінг з описом можливостей, кнопка "Розпочати"

### Екран 2 — Onboarding (4 кроки)
**Крок 1 — Базова інфо:**
- Ім'я
- Рік народження (автоматично визначає вік, при 40+ — враховує перименопаузу)
- Поточний день циклу (з live-preview фази)

**Крок 2 — Цикл:**
- Середня довжина циклу (21-35 днів)
- Тривалість менструації (2-7 днів)
- Тип контрацепції (без / гормональні таблетки / гормональна спіраль / мідна спіраль / бар'єрний / інше)

**Крок 3 — Спосіб життя:**
- Тип харчування (всеїдна / вегетаріанка / веганка / кето / без глютену)
- Рівень фізичної активності (4 варіанти)
- Поточні добавки (магній, D3, омега-3, B вітаміни, залізо, колаген, ашваганда, пробіотики)
- Час сну і підйому

**Крок 4 — Цілі:**
- Більше енергії / Кращий сон / Менше стресу / Гормональний баланс / Здорова шкіра / Мозкова ясність / Довголіття / Здорова вага

### Екран 3 — Daily Check-in (9 кроків)
1. Рівень енергії (слайдер 1-10, кольоровий з emoji)
2. Якість сну (слайдер 1-10)
3. Години сну (кнопки 4-10 год)
4. Настрій (5 emoji-варіантів)
5. Рівень стресу (слайдер 1-10, інвертований)
6. Вода сьогодні (слайдер 0-10 склянок)
7. Рух сьогодні (так/ні + тип: йога/прогулянка/біг/тренування/плавання/велосипед/танці/розтяжка)
8. Симптоми (мультивибір: головний біль / здуття / спазми / мозковий туман / тривожність / втома / дратівливість / нудота)
9. Добавки прийняла (так/ні)

### Екран 4 — AI Інсайт
- Персоналізоване привітання на основі загального Wellness Score
- До 5 карток з інсайтами (адаптовані під вік, фазу циклу, симптоми, цілі, тип харчування, воду, рух)
- Порада дня по фазі циклу
- Перехід до дашборду

**Логіка AI-інсайтів враховує:**
- Фазу циклу (менструальна / фолікулярна / овуляція / лютеальна)
- Вік і перименопаузу (40+)
- Гормональну контрацепцію
- Тип харчування (окремі поради для веганок)
- Добавки з профілю
- Симптоми дня
- Воду і рух
- Цілі користувача

**Wellness Score:** формула (energy + mood×2 + (10-stress) + sleep) / 5

### Екран 5 — Dashboard
- Профіль-резюме (вік, харчування, активність, контрацепція, цілі)
- Поточна фаза циклу з детальним описом, рухом і харчуванням
- Статистика останнього check-in (4 метрики + симптоми)
- Графік динаміки за 7 днів (AreaChart: енергія + сон)
- Історія попередніх check-inів

## 🎨 Дизайн

- **Назва:** VIVE ✦

## ⚠️ Важливі обмеження MVP

1. **Немає Claude/OpenAI API** — AI-інсайти це "розумна логіка" на if/else, не справжній AI. Потрібен API ключ для реального AI.
2. **Дані зберігаються в пам'яті** — при перезавантаженні сторінки все скидається (немає бази даних, немає localStorage).
3. **Джерела даних** — рекомендації базуються на загальних дослідженнях (Alisa Vitti "In the FLO", Sarah Gottfried, Peter Attia "Outlive"). Не є медичною порадою — потрібен дисклеймер.
4. **Норма води** — зараз 7-8 склянок (ВООЗ), не персоналізована під вагу/активність.
